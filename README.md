# init-my-libs

 یک ابزار CLI برای توسعه‌دهندگان React است که نصب و بروزرسانی کتابخانه‌های محبوب React را در پروژه‌ها ساده می‌کند و وقت شما را ذخیره می‌کند

## ویژگی‌ها

- انتخاب تعاملی کتابخانه‌ها برای نصب
- بررسی نصب قبلی کتابخانه‌ها
- پرسش قبل از بروزرسانی کتابخانه‌های قدیمی
- نمایش پیشرفت نصب با Spinner
- نمایش توضیح کوتاه انگلیسی برای هر کتابخانه
- قابل اجرا در ریشه هر پروژه Node.js

## کتابخانه‌های پشتیبانی شده

- SweetAlert2 نمایش alert و popup زیبا
- React Toastify نمایش نوتیفیکیشن در React
- Framer Motion انیمیشن در React
- AOS (Animate On Scroll) انیمیشن هنگام اسکرول
- Yup (Validation) کتابخانه اعتبارسنجی داده‌ها
- TanStack Table جدول قدرتمند برای React
- Axios کلاینت HTTP برای API
- Day.js کتابخانه تاریخ و زمان
- Lucide React آیکون‌های SVG زیبا برای React
- Tippy.js کتابخانه tooltip و dropdown
- FilePond (File Upload) آپلود فایل
- React ApexCharts نمودارهای React
- Tabler Icons React آیکون‌های متن‌باز
- React Beautiful DnD کشیدن و رها کردن عناصر
- CountUp.js نمایش شمارنده انیمیشنی

## نصب

نصب به صورت global

npm install -g init-my-libs

یا اجرای مستقیم با npx بدون نصب

npx init-my-libs

## استفاده

CLI را در ریشه پروژه خود اجرا کنید

init-my-libs

- ابتدا لیست کتابخانه‌ها را مشاهده و انتخاب کنید
- اگر کتابخانه‌ای نصب شده بود از شما پرسیده می‌شود آیا مایل به بروزرسانی آن هستید یا خیر
- پس از انتخاب همه کتابخانه‌های تایید شده در پوشه node_modules پروژه نصب می‌شوند

## نکات مهم

- مطمئن شوید CLI در ریشه پروژه اجرا می‌شود تا کتابخانه‌ها در همان node_modules نصب شوند
- برای بروزرسانی کتابخانه‌های قدیمی CLI قبل از اجرای نصب از شما سوال می‌پرسد
